Setup (DEV)
===========
    git clone https://github.com/alexnormand/dinoipsum.git
    cd dinoipsum
    npm install && bower install
    grunt

Setup (PROD)
============
    git clone https://github.com/alexnormand/dinoipsum.git
    cd dinoipsum
    npm install && bower install
    grunt build
    cd dist/
    npm install --production
    node app.js
